## Lambda Shell implementation by Sergio Delúquez González and Santiago Arboleda### compiling informationCompiled using:+ GHC 9.12.1 (latest stable release on Mac)+ Cabal 3.10.3.0+ MacOS Sequoia 15.1.1+ ASR’s LambdaShell Fork (version 0.9.9.1)### Notes+ The program is named Baren_hw.lam, and should be executed by loading it on LambdaShell with the :load command (asuming is on the same directory).+ examples are named as succ0, succ2, pred0, pred1, is02 and is00.+ The examples are already built in the code, in the examples heading. However feel free to evaluate the code with others as well.+ help from ChatGPT was used in order to overcome parser restrictions.+ code was taken from prelude.lam in order to add some definitions (shown in the code).
